# Commercial Use Policy

SavannaGuard Community edition is source-available.

Allowed without separate agreement:
- Self-hosting for personal use
- Self-hosting for internal company use
- Modifying code for your own environment

Requires separate commercial agreement:
- Offering SavannaGuard as a hosted/managed service for third parties
- Reselling/white-labeling a competing bot-protection SaaS based on this code

For managed and enterprise licensing, use a separate commercial contract.
