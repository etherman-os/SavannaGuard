# Support

SavannaGuard Community is a self-hosted, source-available project.

## Community Support

Use GitHub issues for reproducible bugs, documentation problems, and focused
feature requests. Include the version or commit, deployment method, exact config
shape, and sanitized logs.

## Security Issues

Do not open a public issue for sensitive vulnerabilities or working bypasses.
Follow [SECURITY.md](SECURITY.md).

## Commercial Use

Self-hosting for personal and internal company use is allowed under the
repository license. Hosted/managed resale requires a separate commercial
agreement; see [COMMERCIAL_USE.md](COMMERCIAL_USE.md).
